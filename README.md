# Yajir（ヤジール）

超小型MCU向けの、小さな組み込みスクリプト言語のリファレンス実装（純C）。

**「すべてはポート。`値 -> ポート` で値を流す」** ── この一つの発想だけで、
GPIO・UART・タイマ・文字列処理・イベントを、同じ記法で扱います。

```
MAIN
    1 -> LED1        // 点ける
    500 -> WAIT      // 500ms 待つ
    0 -> LED1        // 消す
    500 -> WAIT
END

ON BTN               // ボタン押下イベント
    "pushed!" -> STDOUT
END
```

- **ヒープ不使用** ── 固定RAMアリーナ1本だけで動く（`malloc` なし）
- **スタック型バイトコードVM** ＋ ワンパスコンパイラ ＋ 協調tick
- **ホスト非依存のコア** ── ボード側は「ポートを register するグルー」を差し替えるだけ
- 端末からスクリプトを貼り付け → その場でコンパイル＆常駐実行

## フットプリント（実測）

| | |
|---|---|
| **RAM** | 約 **12 KB**（VMアリーナ1本・ヒープ不使用。実機ローダは受信バッファ込みで約16 KB） |
| **Flash** | 約 **24 KB**（Cortex-M4 コア一式。デバッグビルド実測、最適化でさらに小さく） |
| **動作実績** | STM32 NUCLEO-L476RG（Cortex-M4）── Lチカ / ボタン / フィボナッチ ほか |

バイトコード4 KB・文字列プール1 KB・スロット/ポート数などは `src/core/script_config.h` で調整できます。

## 収録物

- `src/core/` ── 言語コア（VM・コンパイラ・スケジューラ・文字列ユーティリティ）
- `src/host/` ── PC（Windows）ホストと STM32 実機ホスト
- `tests/`    ── フェーズ別ユニットテスト
- `docs/` / `scripts/` ── 組み込みポート早見表とサンプルスクリプト（`.yaj`）
